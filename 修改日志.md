# 修改日志

## 2026-06-13 — 新增开屏 Loading 动画 + 鼠标跟随效果

### 修改概述
1. 为项目添加页面打开时的 Loading 动画效果，使用 GSAP 驱动的分段方格进度条，动画结束后淡出显示实际内容。
2. 添加鼠标跟随 👁 效果，Loading 结束后显示，平滑跟随鼠标指针。

### 新增文件
- **`app/components/LoadingScreen.tsx`** — 开屏 Loading 动画组件
  - 全屏覆盖层，深色背景 (`#0f172a`)
  - Cyan 发光 "Loading" 标题（沿用项目 `.digital-clock` 风格）
  - 32 段方格进度条，GSAP timeline 驱动填充动画（总时长约 1.7s）
  - 响应式适配（桌面 12x12px 方格，移动端 6x6px 方格，始终单行）
  - 动画完成后淡出并通过 `onDone` 回调移除

- **`app/components/MouseStalker.tsx`** — 鼠标跟随组件
  - 使用 👁 emoji（44px）跟随鼠标指针
  - 平滑跟随（LERP=0.18），偏移右下 22px
  - 仅在桌面端（fine pointer）显示，移动端自动隐藏
  - 使用 `requestAnimationFrame` 驱动跟随循环，空闲时停止

### 修改文件
- **`app/root.tsx`** — 集成 LoadingScreen + MouseStalker
  - 新增 `loading` state 控制 loading/内容切换
  - 使用 `React.lazy` + `Suspense` 懒加载 LoadingScreen，兼容 SSR
  - Loading 结束后同时渲染 `<MouseStalker />` + `<Outlet />`

- **`package.json`** — 新增 `gsap` 依赖

## 2026-06-16 — 面板弹出动画 + 鼠标视差效果

### 修改概述
1. 为仪表盘 5 个面板添加弹簧弹出动画（scale 0.8 → 1，elastic.out，弹 3 次），从左到右、从上到下依次交错 100ms 播放。
2. 鼠标悬浮面板时产生反向视差位移（最大 6px），离开时弹性复位。

### 新增文件
- **`app/components/AnimatedPanel.tsx`** — 通用动画面板容器组件
  - `index` prop 控制交错延迟（`index × 100ms`）
  - GSAP `elastic.out(1, 0.5)` 弹簧弹出（scale + opacity）
  - 鼠标视差：`quickTo` 驱动 X/Y 偏移，方向与鼠标移动相反，最大 6px
  - 鼠标离开：弹性复位 `elastic.out(1, 0.4)`
  - 仅桌面端启用视差效果

### 修改文件
- **`app/routes/home.tsx`** — 面板接入 AnimatedPanel
  - 折线图面板 → `<AnimatedPanel index={0}>`
  - 时钟卡片 → `<AnimatedPanel index={1}>`
  - 机器人状态卡片 → `<AnimatedPanel index={2}>`
  - 最后发言卡片 → `<AnimatedPanel index={3}>`
  - 日历热力图面板 → `<AnimatedPanel index={4}>`

### 后续调整
- **交错间隔**：100ms → 500ms → **250ms**（终值）
- **初始延迟**：增加 0.3s 基础延迟，避免与 Loading 淡出重叠
- **视差最大位移**：6px → **10px**
- **修复 Nivo 图表尺寸问题**：弹出动画完成后派发 `resize` 事件，强制 ResponsiveLine/ResponsiveCalendar 重新计算尺寸
- **修复鼠标进入抖动**：添加 `readyRef` 标志位，弹出动画未完成前阻止视差响应；`quickTo` 改为 `gsap.to` + `overwrite: "auto"` 避免残留 tween 干扰
